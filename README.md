# Productivity Telegram Bot

Мой первый Telegram-бот на Python.

## Возможности

* 💡 Случайная идея
* 🔥 Мотивация
* 📅 План на день
* 💰 Идея заработка
* 📊 Статистика

## Команды

* `/start` — открыть меню
* `/idea` — получить случайную идею
* `/motivation` — получить мотивацию
* `/money` — получить идею заработка
* `/plan` — показать план на день
* `/stats` — показать статистику

## Технологии

* Python
* python-telegram-bot

## Запуск

1. Установить зависимости:

```bash
pip install python-telegram-bot
```

2. Вставить токен бота в переменную `TOKEN`

3. Запустить:

```bash
python bot.py
```
